ADI 6 - Service API

Endpoints

  • /image/list = List all images (limited to 25 in each request)
    • page (optional, default 1) = page number
    • limit (optional, default 25, max 50) = page number
    • tags (optional) = select only images that contains the specified tags
    Example: /image/list?tags=(c)touhou&page=3
  • /image/000 = Get a single image with ID 000 Example: /image/10200
